# JavaScript Array Basics

A beginner-friendly JavaScript project that demonstrates different looping techniques, object iteration, and commonly used array methods such as `map()`, `filter()`, and `reduce()`.

## 📌 Overview

This repository contains practical exercises designed to strengthen fundamental JavaScript concepts. The tasks focus on traversing arrays, working with objects, and manipulating data using built-in array methods.

## 🚀 Topics Covered

* Arrays
* Objects
* for Loop
* forEach()
* for...of Loop
* for...in Loop
* map()
* filter()
* reduce()

## 📚 Tasks Completed

### 1. Looping Through an Array

Print array values using:

* for loop
* forEach()
* for...of loop
* for...in loop

### 2. Loop Through an Object

Display object keys and values using a for...in loop.

### 3. Using map()

Create a new array by subtracting 10 from each element.

### 4. Using filter()

Filter and return numbers greater than 10.

### 5. Using reduce()

Calculate the total sum of all values in an array.

## 🎯 Learning Outcomes

By completing these exercises, I learned:

* Different ways to iterate through arrays
* How to work with JavaScript objects
* Data transformation using map()
* Data filtering using filter()
* Data aggregation using reduce()
* Writing clean and efficient JavaScript code
